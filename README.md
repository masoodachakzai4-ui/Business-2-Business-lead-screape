## Project Overview

B2B Lead Scraper is an AI-powered tool designed to help businesses find and collect potential leads from Google Maps using simple search queries.

The tool automates the lead generation process by collecting business information and organizing data, helping companies save time and improve their marketing efforts.

## Features

- 🔍 Search businesses using simple queries
- 🤖 AI-powered lead collection
- 📍 Extract business information from Google Maps
- 📊 Automatically organize collected leads
- ⏱️ Saves time in manual lead research
- 📁 Export and manage lead data easily

## How It Works

1. User enters a search query (for example: "restaurants in Karachi").
2. The system searches for relevant businesses.
3. Business details are collected automatically.
4. Data is organized into a structured format.
5. Users can use the leads for business outreach and marketing.

## Technologies Used

- AI Automation

- JavaScript / APIs
